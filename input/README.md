# Sources documentaires

Ce dossier est l'unique point d'entrée du corpus.

- Les documents à la racine de `input/` appartiennent au corpus de thèse actuel.
- Les documents dans `input/old_docs/` sont des archives et anciens projets.
- Les cartes distillées et les registres de traitement sont dans `cartes/`.
- Les rendus, extractions temporaires et résultats de pipeline ne doivent pas être
  ajoutes ici.

Le statut de lecture de chaque source est maintenu dans
`cartes/REGISTRE_TRAITEMENT.md`, et l'ordre de traitement dans
`cartes/ORDRE_TRAITEMENT_DOCUMENTS.md`.
