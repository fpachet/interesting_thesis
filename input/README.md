# Sources documentaires

Ce dossier est l'unique point d'entree du corpus.

- Les documents a la racine de `input/` appartiennent au corpus de these actuel.
- Les documents dans `input/old_docs/` sont des archives et anciens projets.
- Les cartes distillees et les registres de traitement sont dans `cartes/`.
- Les rendus, extractions temporaires et resultats de pipeline ne doivent pas etre
  ajoutes ici.

Le statut de lecture de chaque source est maintenu dans
`cartes/REGISTRE_TRAITEMENT.md`, et l'ordre de traitement dans
`cartes/ORDRE_TRAITEMENT_DOCUMENTS.md`.
